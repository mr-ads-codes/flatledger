# Enable FlatLedger on GitHub Pages

1. Open <https://github.com/mr-ads-codes/flatledger>.
2. Select **Add file** and then **Upload files**.
3. Upload `index.html` from this folder into the repository root.
4. Commit directly to the `main` branch.
5. Keep **Settings → Pages → Deploy from a branch → main → /(root)** selected.
6. Wait one or two minutes, then open <https://mr-ads-codes.github.io/flatledger/>.

The visible website is hosted on GitHub Pages. Its API uses the existing Cloudflare D1 database so all members share the same expenses.
